# FinTrack AI (Octoverse Hackathon)

Projeto simples de **landing page + mini app** para controle financeiro com insights de IA (simulados).

## ✨ Funcionalidades
- Adicionar gastos por categoria
- Ver total acumulado
- Identificar categoria com maior gasto
- Receber recomendação automática baseada no padrão de gastos
- Persistência local com `localStorage`

## 🧰 Stack
- HTML
- CSS
- JavaScript puro

## 🚀 Como rodar
1. Abra o arquivo `index.html` no navegador.
2. Adicione gastos e veja os insights.

## 🏆 Alinhamento com o hackathon
- **Fintech**: gestão de despesas
- **Machine Learning/AI**: recomendação inteligente (simulada)
- **Enterprise**: dashboard simples para tomada de decisão
